# Commands

## Upgrading Flutter
```bash
flutter upgrade
```

## Upgrading Dart on MacOS
```bash
brew install dart
brew upgrade dart
```

## Getting Current Versions
```bash
flutter --version
dart --version
```

## Creating a New Dart Project
```bash
dart create YOUR_PROJECT_NAME
dart create . --force
```
